# BlackDPI

BlackDPI — это модифицированный форк [ByeDpiLight](https://github.com/DroidProger/ByeDpiLight)для Android, предназначенный для запуска [ByeDPI](https://github.com/dovecoteescapee/byedpiandroid) в режиме VPN. Проект сохраняет базовую логику оригинального приложения, но добавляет дополнительные настройки интерфейса и работы с параметрами.

Проект лицензирован под GNU GPL v3. Как модифицированный форк [ByeDpiLight](https://github.com/DroidProger/ByeDpiLight) и производный проект на основе наработок оригинального [ByeDPI](https://github.com/dovecoteescapee/byedpiandroid), BlackDPI распространяется с соблюдением принципов copyleft: любые дальнейшие изменения и распространения должны сохранять те же свободы для пользователей и публиковаться под той же лицензией.

Об аргументах командной строки можно прочитать в readme [byedpi](https://github.com/hufrea/byedpi).
В настройках hev-socks5-tunnel необходимо указывать такой же порт как в настройках byedpi. Dns можно указать любой, обычно это 1.1.1.1 или 8.8.8.8.
